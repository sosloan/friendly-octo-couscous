HOW THIS WORKS
--------------
1. Pick any prompt below. Copy it. Paste it. Hit send. 1-click and go.
2. After you read the response, look at the four NEXT PROMPTS.
3. Pick ONE -- or pick MULTIPLE and combine them (e.g. [A]+[C]).
4. Add [-] PIN IT DOWN to any combo to tighten scope and shrink output.
5. [-] PIN IT DOWN used ALONE is your stop/reset signal.

MULTI-SELECT EXAMPLES:
  [A]       -> Scale up only.
  [A]+[B]   -> Scale up AND go deeper at the same time.
  [B]+[C]   -> Go deeper AND shift the role perspective.
  [A]+[D]+[-] -> Scale up, add adjacent domain, but keep output tight.
  [-]       -> Pull back hard. Strip to the minimum. Pin the scope.

THE 4 SUGGESTION TYPES (appear after every prompt):
  [A] SCALE UP    = Bigger scenario. More agencies, stakes, or complexity.
  [B] GO DEEPER   = More detail, nuance, or rigor on the same task.
  [C] SHIFT ROLE  = Same scenario, different responder's perspective.
  [D] ADJACENT    = Neighboring domain, same format and difficulty.
  [-] PIN IT DOWN = Constrain. Shorter output. Fewer assumptions. Tighter scope.
                    Use alone to reset. Combine with [A]-[D] to focus the combo.

 GALLERY NAVIGATION RULES
================================================================================

  PICK -> RESPOND -> CHOOSE NEXT -> REPEAT

  After every response, four suggestions appear:

    [A] SCALE UP    = More agencies, higher stakes, bigger scope.
    [B] GO DEEPER   = More detail, rigor, or nuance on the same task.
    [C] SHIFT ROLE  = Same scenario, different responder's perspective.
    [D] ADJACENT    = Neighboring domain, same format and difficulty.
    [-] PIN IT DOWN = Constrain. Use alone to reset. Combine to narrow a combo.

  MULTI-SELECT COMBOS:
    [A]+[B]      -> Bigger AND more detailed.
    [B]+[C]      -> Deep dive from a different seat.
    [A]+[D]      -> Level up AND move to adjacent domain.
    [A]+[B]+[-]  -> Level up and go deep, but keep the output tight.
    [-]          -> Stop climbing. Strip to minimum. Find the floor.

  WHEN DOES THE GALLERY UPDATE?
    * Any [A] selection unlocks the next Gallery tier for that prompt.
    * [A]+[A]+[A] in sequence on any row -> Gallery 1 row promoted to Gallery 0.
    * [-] alone on any prompt -> that tile resets to its simplest form.
    * Multi-select combos generate hybrid prompts not listed here --
      those are Gallery 2+ territory, generated on-the-fly.

 FEED-FORWARD SYSTEM SUMMARY
================================================================================

  Each prompt is a node. Each response is an edge. The gallery is a graph.

  With 4 options per node (plus multi-select combos), the branching factor
  is effectively unlimited. You never hit a dead end.

    [A] SCALE UP    -> New node at higher complexity.
    [B] GO DEEPER   -> New node at greater depth, same altitude.
    [C] SHIFT ROLE  -> New node at same complexity, different seat.
    [D] ADJACENT    -> New node in a neighboring operational domain.
    [-] PIN IT DOWN -> Constrains the next node. Alone: reset to minimum.

  MULTI-SELECT creates composite nodes not in this list.
  Those are Gallery 2+ -- generated from your actual responses.

  Starting nodes in Gallery 0:   A1, A2, A3, B1, B2, B3, C1, C2, C3
  Gallery 1 nodes:               A4, B4, C4
  Gallery 2+:                    Generated from [A] chains or multi-select combos

  The Sheriff's team can run this like a menu.
  Pick one. Combine. Pin it down when you've found the depth you need.
  The prompts are already written. No prompt engineering required.
  1-click and go.
